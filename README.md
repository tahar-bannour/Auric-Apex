## Installation & Usage

1. Open **TradingView** → **Pine Editor**.
2. Copy the file `super_vision_signal_v5.pine` into the editor.
3. Click **Add to Chart**.
4. Adjust settings in the input panel:
   - Enable/disable filters
   - Adjust table display preferences
